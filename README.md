# formlang
[![Build Status](https://api.travis-ci.org/tsarn/formlang.svg?branch=master)](https://travis-ci.org/tsarn/formlang)

First of all, compile antlr grammar (needs `antlr4` installed):

    $ ./prepare.sh

Usage:

    $ pipenv run python -m formlang <arguments>

Testing:

    $ pipenv run pytest
